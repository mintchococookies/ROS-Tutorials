ROS

  • pwd shows you which directory/path you're currently in.
  • apt-get is for installing things from the internet
  • sudo means superuser do - requires your password (admin functionality)
  • crtl+c to stop the running terminal
  • everytime you open a new terminal you should enter (to source ROS) *only applies to the first terminal you open.
$ source /opt/ros/melodic/setup.bash

# This is bc we installed .bash instead of .bashrc
  • to enter the catkin workspace you should enter (also when you open a new tab)
$ . ~/catkin_ws/devel/setup.bash

# These just sources the setup.bash file which gets read and configures the terminal for us.
  • roscore needs to be run at the start of the ROS terminal in order to run nodes.
  • to get to the scripts folder within a package just 
$ cd scripts
  • To make a node executable:
$ chmod +x add_two_ints_server.py
